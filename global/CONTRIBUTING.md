# Contributing to the ICAIRE Global AI Readiness Monitor

Thank you for your interest in contributing! This project aims to be a living, open resource for policymakers, researchers, and organizations worldwide.

- When a country completes a new RAM assessment or a new index is published, submit a PR updating the relevant JSON file in `data/`
- Include the source URL and date in your PR description
- Follow the existing JSON schema for each data source

### üêõ Bug Reports
- Open an issue describing the problem
- Include screenshots if it's a visual bug
- Note which browser/device you're using

### üí° Feature Requests
- Open an issue with the "enhancement" label
- Describe the feature and why it would be useful
- If possible, include mockups or examples

### üìä New Data Sources
- Propose new indices or data sources via issues
- Include: source name, URL, coverage, methodology, and license/terms
- We prioritize openly licensed, regularly updated, country-level data

### üåç Translations
- We welcome Arabic, French, and Spanish translations
- Translation files should follow the existing structure

## Data Quality Standards

- All data must include source attribution
- Estimated/derived values must be clearly marked
- Include the year/version of the source data
- Prefer official/primary sources over secondary reporting

## Code Standards

- Keep it simple: vanilla HTML/CSS/JS, no build tools
- Follow existing code style
- Test in Chrome, Firefox, and Safari
- Ensure responsive design works on mobile

## Contact

For questions: info@icaire.org

## Code of Conduct

Be respectful, constructive, and inclusive. This project serves a global audience across cultures and political contexts. We follow UNESCO's principles of dignity, equality, and mutual respect.
